# MemCache - Mac Menu Bar Meeting Reminder For My Old Age

A minimal, elegant Mac menu bar application that keeps your upcoming meetings visible and accessible without cluttering your workflow.

## Project Overview

This repository contains the project specification and development roadmap for a macOS menu bar widget that displays upcoming calendar meetings with smart urgency indicators.

## Documentation

- **[MenuBarMeetings.md](MenuBarMeetings.md)**: Complete project specification including features, technical requirements, and implementation phases
- **[MenuBarMeetings-Sprints.md](MenuBarMeetings-Sprints.md)**: Detailed 8-sprint development plan with tasks, deliverables, and timelines

## Key Features

- **Minimal Menu Bar Widget**: Shows next meeting time + brief title
- **Smart Urgency Indicators**: Visual changes as meetings approach (<30 minutes)
- **Expandable Daily View**: Click to see full day's meeting schedule
- **Universal Calendar Support**: Apple Calendar, Google Calendar, Microsoft Outlook
- **Native macOS Integration**: Proper system theme support and performance

## Development Timeline

8 sprints (12-16 weeks) structured for iterative development:
- **Sprints 1-2**: Foundation and basic calendar integration
- **Sprints 3-4**: Core functionality and smart display
- **Sprints 5-6**: Multi-calendar support and advanced features  
- **Sprints 7-8**: Final polish and production readiness

## Target Users

Mac users who attend multiple daily meetings and want a distraction-free way to stay aware of their schedule without constantly checking calendar apps.
